相对估值法和三档建仓

2020-02-28 股票 估值法, 建仓 评论

相对估值法

运用市盈率 PE 与净利润的变化来给股票估值，适用于盈利稳定、成长性良好的公司。

三档建仓

  • 第一档：建仓 PE = 1.21 * 历史最低 PE
  • 第二档：加仓 PE = 1.1 * 历史最低 PE
  • 第三档：重仓 PE = 历史最低 PE

找到盈利稳定的公司在市盈率 PE 较低时买入，查看【历史最低 PE】，从理杏仁输入代码—基本面查看 PE-TTM

举例：伊利股份当前 PE = 30.22，股价 27.3 元，历史最低 PE = 18.35

第一档

建仓：PE = 1.21 * 18.35 = 22.2 倍

建仓 PE / 当前 PE = 22.2 / 30.22 = 0.73

建仓价格：0.73 * 27.3 = 19.93 元

第二档

加仓：PE = 1.1 * 18.35 = 20.19 倍

加仓 PE / 当前 PE = 20.19 / 30.22 = 0.67

加仓价格：0.67 * 27.3 = 18.29 元

第三档

重仓：PE = 18.35

重仓 PE / 当前 PE = 18.35 / 30.22 = 0.61

重仓价格：0.61 * 27.3 = 16.65 元

切换原理

半年或一年重新筛选，不符合的卖出，再补入符合的股票。

卖出原则：

  1. 发现分析有误，及时卖出；
  2. 不再符合选股条件（ROE < 15% 或 PB > 8）；
  3. PE 超过中位值（灰线）分三档卖出：
    • PE = 中位值，卖出总资金的 1/3；
    • PE = 中位值 * 1.1，卖出此时资金的 1/2；
    • PE = 中位值 * 1.21，全部卖出。
