白马股，不要怂就是干

2020-02-25 投资 股票 评论

白马股是什么？

白马股，是指长期绩优、回报率高并具有较高投资价值的股票。市场表现大多为持久的慢牛攀升行情，有时业表现出较强的升势。

白马股指标

市盈率 PE = 市值 / 净利润

市净率 PB = 市值 / 净资产

股息率 = 年度分红金额 / 派息时的股价

判断方法：

  • 连续七年或连续十年净资产收益率 ROE >= 15%
  • 0 < 市净率 PB < 8
  • 剔除周期股

哪些属于强周期行业

  • 大宗商品行业：石油、有色金属、煤炭、钢铁、水泥等
  • 远洋运输行业：造船、港口机械
  • 房地产行业
  • 非生活必须品：汽车行业

注意：如果一支股票近十年的 ROE，最高与最低相差在 10% 以上，且至少有一年 ROE 为负数，极有可能是周期股。

识别“困境白马”

由以上白马股筛选出来的股票后，还要特别注意是否包含“困境白马”。

判断方法：单季度同比业绩增速明显下滑的公司，业绩增速包括营收增速和净利润增速，特别注意：分为点在 10% 以下的，困境白马的可能性极大。
