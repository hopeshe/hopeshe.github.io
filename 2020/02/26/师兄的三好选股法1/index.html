师兄的三好选股法1

2020-02-26 投资 股票 评论

三好选股法（3 + 1）

  • 好行业
  • 好公司
  • 好价格
  • 好管理、好时机等

1.好行业

技巧：看行业的历史回报

方法：中证指数官网—指数系列—中证行业指数

举例：300 能源和 300 医药，选择成分股数量小一些的。

操作：打开【理杏仁】看它的历史回报：

  • 【300 能源】近十年年化涨跌幅 -9.66%，成立以来年化涨跌幅 -8.20%
  • 【300 医药】近十年年化涨跌幅 7.27%，成立以来年化涨跌幅 8.99%
  • 【300 消费】近十年年化涨跌幅 13.38%，成立以来年化涨跌幅11.39%
  • 【中证白酒】近五年年化涨跌幅 25.18%，成立以来年化涨跌幅 24.49%

工具：【可比公司】

例：五粮液

打开可比公司，输入“五粮液”，按市值排名查看行业前三的投资回报率。

影响好行业的两个因素

  • 需求（只有这个行业的蛋糕越来越大有钱可赚，进来赚钱的人才会多，行业才会繁荣）
  • 竞争 （进来赚钱的人要有竞争力，公司要有护城河、规模优势来保证自己能够赚到更多的钱，公司能赚到更多的钱，投资人自然也能从投资中获利）

好行业要从需求和竞争中分析

什么是需求？

一个行业的营收空间（市场规模、营收规模、销售额、体量）

需求的五种态势

  1. 快速增长 增速 >= 15%，锂电池
  2. 稳健增长 增速（5%-15%），防水材料
  3. 稳定 增速（0-5%），乳制品
  4. 衰退 负增长，缝纫机
  5. 波动 增长和衰退交错，周期股（周期之王：券商股）

好行业：：前三的需求态势都可以，分析一个行业未来五到十年的趋势

工具

  • 萝卜投研
  • 券商研报
  • 雪球
  • 百度
  • 微信公众号

举例：双汇发展

方法：萝卜投研，搜“肉制品 [关键词]” 肉制品 市场规模

券商研报—图表，得出【稳定】的需求态势

搜“肉制品 消费”，还是会【稳定】增长

萝卜投研可搜索的关键字

行业词 + 行业规模、消费、市场规模、市场增速、增速

收入 = 数量 * 单价

两个因子：单价、数量

可从乳制品、白酒、调味品等行业先入手了解。

行业生命周期

行业生命周期 导入期 成长期 成熟期 衰退期
公司数量 少 增加 减少 减少
行业增速 波动大 高增长 个位数增长 负增长
经营风险 大 大 降低 增大
行业利润 小 大量增加 稳定 减少

在成长期结束，成熟期开始之前买入

缝纫机【衰退】、肉制品【稳定】、乳制品【稳定】、高端白酒【稳定】、煤炭【衰退】、锂电池【快速增长】、光电子元器件【？

投资的原则：能力圈原则、艺术模糊原则

总结：判断一个行业是否符合标准，要求营收空间大（需求大），门槛高（竞争小）

竞争：服装行业，过去有【稳定】增长，但中国没有服装的牛股，

看森马，美特、海澜之家

电影票房：【快速增长】，竞争大，不出牛股。都不符合 {好行业}

从下列行业中挑一个，看它的需求态势是哪种类型?

  • 烟草
  • 快递
  • 卤味食品
  • 寿险和健康保险
  • 糖尿病药物
  • 网络游戏
  • 名牌箱包

竞争

1.产业分蛋糕的人

  • 供应商（员工）
  • 同行厂家
  • 政府收税
  • 股东

2.侠义竞争：同行厂家

3.竞争对手

  • 现有同行
  • 新进入者

4.最好的竞争格局：现在有序，进入门槛高。

5.现在有序

  • 垄断 一家独大 芯片：英特尔、AMD 【没有竞争】
  • 寡头垄断 若干家独大 通信：移动、联通、电信 【轻度竞争】
  • 垄断性竞争 只有几个大玩家竞争 + 小玩家 保险行业 【中度竞争】
  • 完全竞争 充分竞争（无序） 【激烈竞争】

怎么看？？

  • 行业集中度和行业竞争格局
  • 行业主要公司的回报率

6.进入门槛高

特殊资源：专利技术、资质牌照、工艺

品牌

规模效益

操作步骤（使用罗卜投研）

  1. 找到所在的行业，关心的公司
  2. 查找行业的竞争格局（数量、集中度）
  3. 查找行业主要公司回报率
  4. 结合常识和知识进行判断

要前三种竞争状态，注意行业是动态的

总结

什么才是好行业？需求不衰退，竞争不激烈
