师兄的三好选股法1

2020-02-26 股票 三好选股法 评论

三好选股法（好行业 + 好公司 + 好价格 + 加分项：好管理、好时机等）

一、好行业

1.看该行业的历史回报率

方法：中证指数官网—指数系列—中证行业指数—行业代码—理杏仁查看历史回报率，如果不知道所选公司属于哪个行业，可用【可比公司网站】查看。

举例：300 能源和 300 医药，选择成分股数量小一些的。

  • 【300 能源】近十年年化涨跌幅 -9.66%，成立以来年化涨跌幅 -8.20%
  • 【300 医药】近十年年化涨跌幅 7.27%，成立以来年化涨跌幅 8.99%
  • 【300 消费】近十年年化涨跌幅 13.38%，成立以来年化涨跌幅11.39%
  • 【中证白酒】近五年年化涨跌幅 25.18%，成立以来年化涨跌幅 24.49%

2.影响好行业的两个因素（行业需求、行业竞争）

  • 需求（只有这个行业的蛋糕越来越大有钱可赚，进来赚钱的人才会多，行业才会繁荣）
  • 竞争 （进来赚钱的人要有竞争力，公司要有护城河、规模优势来保证自己能够赚到更多的钱，公司能赚到更多的钱，投资人自然也能从投资中获利）

需求（一个行业的营收空间：市场规模、营收规模、销售额、体量）

需求的五种态势（好行业的需求态势：前三个，分析一个行业未来五到十年的趋势）

  1. 快速增长 增速 >= 15%，锂电池、影视娱乐（光线传媒）
  2. 稳健增长 增速（5%-15%），防水材料、乳制品、空调（格力电器）
  3. 稳定 增速（0-5%），肉制品（双汇发展）、高端白酒
  4. 衰退 负增长，代表这个行业的需求在减少。缝纫机、煤炭
  5. 波动 增长和衰退交错，周期股（周期之王：券商股）、原材料（钢材等）

举例

  1. 快速增长：影视行业的【光线传媒】上市八年多，股价从 2.8 元涨到 9 元多，涨幅约 3 倍，年化收益率 12.6%
  2. 稳健增长：空调的【格力电器】，过去十年股价从 4.75 元涨到 58 元，涨幅约 12 倍，年化收益率 28%
  3. 稳定：肉制品【双汇发展】，过去十年股价从 9.2 元涨到 25 元，涨幅约 2.7 倍，年化收益率 10.5%

小结：好行业除了需求的因素还要继续分析【竞争】，以上信息可在【萝卜投研】查找。

行业生命周期 导入期 成长期 成熟期 衰退期
公司数量 少 增加 减少 减少
行业增速 波动大 高增长 个位数增长 负增长
经营风险 大 大 降低 增大
行业利润 小 大量增加 稳定 减少

在成长期结束，成熟期开始之前买入

竞争

竞争的四种态势（好行业的竞争态势：后三个）

  1. 高烈度竞争，钢铁、纺织等
  2. 中烈度竞争，行业分化有龙头或门槛高：乳制品、空调、手机等
  3. 低烈度竞争，产品有差异化且存在行业龙头：高端白酒、糖尿病特效药等
  4. 无竞争，各个城市的自来水厂

【产品差异越大、行业分化存在龙头、进入门槛越高，这样的行业竞争就越少】

方法：（使用罗卜投研）

  1. 找到所在的行业，关心的公司
  2. 查找行业的竞争格局（数量、集中度）
  3. 查找行业主要公司回报率
  4. 结合常识和知识进行判断

总结

TOP1：需求稳定增长/快速增长 + 无竞争/低烈度竞争

TOP2：需求稳定增长/快速增长 + 中烈度竞争

TOP3：需求稳定 + 无竞争/低烈度竞争

TOP4：需求稳定 + 中烈度竞争

以上四种可视为【好行业】
